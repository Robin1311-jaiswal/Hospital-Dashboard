# Recommended Structure and Order

# 1. Poject Title / Headline

Multi-page Power BI dashboard providing insights into patient management, doctor performance, hospital operations, and financial analysis.
Focusing on Delivering insights into hospital efficiency, billing, inventory, and patient management through visual analytics.

# 2. Short Description / purpose

This interactive Power BI dashboard analyzes 700K+ hospital billing data, 3900+ medicine sales, and operational KPIs to support data-driven decision-making. It tracks patient admissions, 73% discharge rate, doctor performance, bed utilization, and financial trends to improve hospital efficiency and resource management.

# 3. Tech Stack
List the key technologies used to build the dashboard.

Example: The dashboard was built using the following tools and technologies:
• 📊 Power BI Desktop – Main data visualization platform used for report creation.

• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.

• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.

• 📝 Data Modeling – Relationships established among tables (
- Patients
- Doctors
- Appointments
- Billing
- Medicines
- Beds
- Departments
- Staff) to enable cross-filtering and aggregation.
  
• 📁 File Format – .pbix for development and .png for dashboard previews.

# 4. Features / Highlights
The best dashboard explanation format. • Business problem • Goal of the dashboard • Walk through of key visuals (briefly!) • Business impact & Insights

# (i) Business Problem
Hospitals generate large volumes of operational, patient, billing, and inventory data, making it difficult to monitor performance, identify inefficiencies, and make data-driven decisions efficiently.
# (ii) Goal of the Dashboard
The goal of this dashboard is to provide a centralized analytics solution for tracking hospital operations, patient management, financial performance, doctor activity, and medicine inventory through interactive visualizations and KPI monitoring.

# (iii) Walkthrough of key Visuals

# • Overview Dashboard –

Key KPIs (Top Section)

Total Billing Amount: 700K+
Patient Count: 30+
Doctor Count: 15+
Staff Count: 20+
Medicine Sales Quantity: 3900+
Patient Discharge Rate: 73%

These KPIs provide a quick overview of hospital operations, financial performance, and patient management efficiency.

• Patient Charges Breakdown (Bar Chart)

Displays revenue generated from different hospital services:

Surgery: ~430K (highest contributor)
Room Charges: ~174K
Tests: ~52K
Doctor Fees, Medicines, Discounts, and Other Charges

This visual helps identify the major revenue-driving services.

 • Patient Charges Breakdown (Bar Chart)

Tracks discharged patients across different months:

Highest discharge observed in May (8 patients)
Monthly discharge trends monitored from May to October

Helps evaluate patient turnover and operational efficiency.

• Medicine Tracking Dashboard (Matrix/Table Visual

Analyzes medicine sales quantity by:

Month
Day of the Week
Medicine Distribution Trends

Supports inventory monitoring and medicine utilization analysis.

# • Patient Analysis Dashboard
• Displays patient demographics, diagnosis details, admission/discharge dates, and medicine usage.
• Total Billing Amount analyzed: 714K+.
• Helps understand patient treatment patterns and hospital service usage.

# • Doctor Performance Dashboard

Tracks:

Doctor commissions (71K+)
Patient spending (714K+)
Appointment schedules
Doctor specialization and availability

Helps analyze doctor performance and contribution to hospital revenue.

# • Bed Utilization Analysis (Bar Chart)


Compares occupancy across:

General Beds
ICU Beds
Private Rooms

Helps monitor hospital resource allocation and room utilization efficiency.

# • Finance Dashboard

Visualizes:

Monthly medicine sales trends
Revenue distribution by service type
Medicine stock vs. sales performance
Average patient age, room charges, and stock quantity

Supports financial analysis and inventory decision-making.

# (iv) Business Impact and Insights
- Identified surgery as the highest revenue-generating service (~430K).
- Enabled tracking of 3900+ medicine sales and inventory utilization.
- Improved visibility into patient discharge trends and hospital efficiency metrics.
- Helped analyze bed occupancy and resource allocation across hospital departments.
- Provided a consolidated view of operational and financial KPIs for better decision-making.
